Logs

Salon de destination et événements à tracer

Activation

Désactive un groupe entier ou des commandes une par une (slash + préfixe). help reste disponible sauf dans les salons marqués « sans commandes ». Les permissions par salon et rôle se configurent dans l’onglet Permissions.

Permissions

Salons autorisés ou interdits, rôles globaux et permissions par catégorie (modération, admin, premium).

Commandes perso

Réponses déclenchées par le préfixe (ex. $règles). Pas de slash Discord pour les perso.

Déclencheur : 1–32 caractères (lettres y compris accents, chiffres, _, -), nom différent d’une commande du bot. Il est enregistré en minuscules.

Mini-guide — texte dynamique dans la réponse

Placeholders disponibles. Les deux écritures sont équivalentes : {guild} ou {{guild}}.

  • {user} — mention de l’auteur
  • {username} — pseudo (sans mention)
  • {user.tag} / {user.id}
  • {channel} — mention du salon — {channel.name} / {channel.id}
  • {guild} ou {server} — nom du serveur
  • {guild.id}, {guild.name}, {members}
  • {date}, {time}, {now} (timestamp Unix)
  • {reply} / {reply.content} — texte du message cité
  • {reply.user} — mention de l’auteur du message cité
  • {{delete}} — supprime le message de commande après envoi (le bot doit avoir « Gérer les messages »)

Exemple : Bienvenue {user} sur {guild} ! {{delete}}

Modération

Antispam automatique + commandes manuelles

Antispam

Détecte les comptes compromis (liens ou images postés sur plusieurs salons). Supprime le message, envoie un MP d’avertissement, puis applique une sourdine Discord (messages + vocal) en cas de récidive.

Commandes manuelles

  • kick — Expulser (KickMembers)
  • ban — Bannir (BanMembers)
  • timeout / untimeout — Sourdine (ModerateMembers)
  • slowmode — Mode lent du salon (ManageChannels)
  • clear — Purge de messages (ManageMessages)

Active le log Antispam dans l’onglet Logs. Les warns sont dans l’onglet Warns.

Warns

Avertissements, sanctions automatiques et historique

Configuration

Sanctions automatiques quand le seuil est atteint. Les warns antispam comptent aussi.

Warns actifs

Charge un serveur pour afficher les warns.

Commandes

  • warn — Enregistre un warn (+ log mod si activé)
  • warns — Liste les warns d’un membre
  • unwarn — Retire un warn par ID ou tous (unwarn all @membre)

Active le log Avertissements dans l’onglet Logs.

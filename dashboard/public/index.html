À quoi sert l’URL du dashboard ?

En temps normal, laisse ce champ vide : tu ouvres la page fournie par npm run dashboard (ex. http://127.0.0.1:3847), donc le navigateur et l’API sont déjà au même endroit.

Remplis l’URL uniquement si tu ouvres le HTML depuis un autre outil (ex. Live Server sur un autre port) : indique alors l’adresse complète du serveur Node qui tourne le dashboard, sans / à la fin.
